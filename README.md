# Etnia Viajes - Travel Agency Website

## 🌐 Live Site

Online at [etniaviajes.com](https://etniaviajes.com)

## ✨ Features

- **Dynamic Travel Packages** - Managed via Supabase CMS
- **Multi-destination Packages** - Support for complex itineraries
- **Google Places Integration** - Hotel images and reviews
- **Destination Galleries** - Unsplash/Pixabay image integration
- **Google Maps Embed** - Hotel location maps
- **WhatsApp Integration** - Direct customer inquiries
- **SEO Optimized** - Meta tags, structured data, semantic HTML
- **Responsive Design** - Mobile-first approach
- **Dark/Light Mode** - Theme support

## 🛠️ Tech Stack

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Animations:** Framer Motion
- **Backend:** Supabase (PostgreSQL)
- **Icons:** Lucide React

## 🌍 Deployment

Optimized for deployment on **Cloudflare Pages**:

1. Connect repository to Cloudflare Pages
2. Set build command: `npm run build`
3. Set output directory: `dist`
4. Configure environment variables
5. Deploy!

## 📁 Project Structure

```
src/
├── components/       # React components
│   ├── home/         # Homepage sections
│   ├── layout/       # Header, Footer, etc.
│   └── packages/     # Package cards & galleries
├── config/           # App configuration
├── hooks/            # Custom React hooks
├── integrations/     # Supabase client
├── lib/              # Utilities
└── pages/            # Route pages
```
